# kittenfriends
Tutorial for React (from Complete Web Developer in 2023: Zero to Mastery, customized from Robofriends project)

## Features

- Lovely kittens
- ChatGPT integration to have conversations with these lovely kittens.

## To run the project

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
